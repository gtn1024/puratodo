# PuraToDo AI Native 构想

> 这是一个头脑风暴文档，记录未来可能的发展方向。

## 核心理念：从"记录工具"变成"思考伙伴"

传统 Todo 是被动的 —— 你输入、它存储。AI Native 的 Todo 应该是**主动参与你的人生规划**。

---

## 一、输入方式的革命

| 传统 | AI Native |
|------|-----------|
| 手动创建每个任务 | 语音/自然语言输入，自动解析 |
| 手动分类、打标签 | AI 自动理解上下文并归类 |
| 手动设置优先级 | AI 根据语义判断重要程度 |

```
用户说：
"明天下午3点和张三开会讨论Q2计划，记得带上上次的PPT"

AI 自动创建：
- 任务：和张三开会讨论Q2计划
- 时间：明天 15:00
- 关联：需要准备 PPT（子任务）
- 可能关联到"张三"这个联系人
```

---

## 二、智能规划与建议

### 1. 智能排程

```
AI 分析你的习惯：
"你通常在上午效率最高，建议把这个重要任务安排在明天上午"
"这个任务已经延期2次了，今天要处理吗？"
```

### 2. 任务拆解

```
输入："完成年度报告"

AI 建议：
├── 收集各部门数据（2天）
├── 整理数据图表（1天）
├── 撰写报告初稿（2天）
├── 审核修改（1天）
└── 提交最终版本
```

### 3. 依赖关系识别

```
"这个任务依赖于'设计稿确认'，但那个还没完成，要调整顺序吗？"
```

## 三、上下文感知

### 1. 从对话/邮件生成任务

```
// 未来可能：连接 Slack/微信/邮件
从消息中提取："@你 下周提交方案"
→ 自动创建任务 + 关联来源
```

### 2. 时间感知提醒

```
"距离截止日期还有3天，但预估需要5小时，今天开始做？"
```

### 3. 地点/情境感知

```
路过超市 → 提醒购物清单
打开电脑 → 提醒工作相关任务
```

## 四、知识管理融合

```
任务不再孤立，而是关联知识库：

"学习 React 18 新特性"
├── AI 推荐相关文档
├── 自动关联之前的笔记
├── 完成后生成学习总结
└── 可以追问 AI 深入理解
```

## 五、具体功能点列表

| 功能 | 描述 |
|------|------|
| **AI Copilot 侧边栏** | "今天我该先做什么？" - AI 分析并建议 |
| **自然语言创建** | 直接用口语输入，AI 解析成结构化任务 |
| **智能回顾** | 周报/月报自动生成："你这周完成了15个任务..." |
| **情绪追踪** | 分析任务完成模式，提示倦怠风险 |
| **目标对齐检查** | "这个任务对你的年度目标有帮助吗？" |
| **会议纪要 → 任务** | 粘贴纪要，自动提取 action items |
| **反拖延心理辅导** | 检测拖延模式，给出个性化建议 |

## 七、差异化定位思考

市面上已有 AI Todo（如 Motion, Reclaim），但它们偏向"自动排程"。

**我们可以做的差异化：**

- **深度任务理解**：不只是时间，理解任务之间的关系
- **长期目标对齐**：帮你思考"为什么做"
- **成长型 AI**：越用越懂你的工作风格
- **隐私优先**：本地模型选项（Ollama + 小模型）

## 八、MVP 建议

如果想开始，最小的切入点：

### 1. AI 任务解析器
- 输入自然语言 → 返回结构化任务数据
- 技术上最简单，价值立即可见

### 2. 智能周报
- 分析一周完成的任务 → 生成总结
- 用户喜欢这种"成果可视化"
